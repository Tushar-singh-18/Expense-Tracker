# Expense Tracker

A simple web app to track your daily expenses.

## Features
- Add expense with name, amount and category
- Delete any expense
- Total updates automatically
- Data saved in localStorage — persists after refresh

## Tech Used
- HTML
- CSS
- JavaScript
- localStorage

## How to Run
1. Clone the repo
2. Open index.html in browser
3. Start tracking expenses

## What I Learned
- DOM manipulation
- localStorage (save and load data)
- Array methods (filter, reduce, forEach)
- Form validation
- Dynamic rendering with JavaScript
